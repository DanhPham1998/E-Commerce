# E-Commerce

> Backend API for E-Commerce application.

## Usage

Rename "config/config.env" to "config/.env" and update value/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm run start
```
